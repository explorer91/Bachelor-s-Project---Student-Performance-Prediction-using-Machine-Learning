# Bachelor-s-Project---Student-Performance-Prediction-using-Machine-Learning
Developed a machine learning system to predict student academic performance. The model analyzes factors like study time and attendance to classify students into performance categories, helping identify those needing support.

## Installation

1. Install Python 3.8+
   - Download from: https://python.org

2. Create Virtual Environment
   python -m venv venv

3. Activate Environment
   - Windows: venv\Scripts\activate
   - Mac/Linux: source venv/bin/activate

4. Install Required Libraries
   pip install pandas numpy scikit-learn matplotlib seaborn joblib ipykernel

## Usage

Run the main project file:
python project.py

Or use Jupyter notebook:
jupyter notebook

## Required Libraries
- pandas
- numpy  
- scikit-learn
- matplotlib
- seaborn
- joblib
- ipykernel

## Project Info
Predicts student performance (High/Medium/Low) using ML algorithms. Processes factors like study time, attendance, and test scores.
